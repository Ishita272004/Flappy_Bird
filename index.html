Flappy Bird Clone - Java Swing

This is a Flappy Bird clone game made using Java Swing and AWT.

The game runs locally on your machine using Java.

Download

You can download the game as a .jar file and run it locally if you have Java installed.

Download Flappy Bird .jar

How to Run

  1. Download the .jar file above.
  2. Make sure you have Java installed on your computer.
  3. Double-click the .jar file or run it from the command line using java -jar Flappy_Bird.jar.
